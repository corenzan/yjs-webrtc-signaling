# Yjs WebRTC Signaling

See https://github.com/yjs/y-webrtc.

## Development

Clone the repository and run:

```sh
npm start
```

This will install the dependencies and start the development server.

## Deployment

Deployment with Docker Compose is recommended.

```sh
docker compose up -d
```
